# 腾讯云使用情况查询

## 〇、描述
此代码适用于快速查询腾讯云资源的使用情况，不需要登陆到腾讯云的控制面板即可获取信息，方便快捷。

本项目对``tencentcloud-sdk-nodejs``包中的api进行调用并选择性输出为``pages/api``

本代码作用：
1. 腾讯云资源快速查询
2. 腾讯云资源监控报警

本代码仅作为个人学习使用


## 一、安装
```
npm install
```

## 二、配置
复制``sample.env``为``.env``并填写里面的内容

## 三、运行
```
npm run start
```
